Special characters  #    pound sign      Denotes start of comment  ;    semicolon       Terminates a statement  ,    comma           Separator in various lists  @    at sign         Identifies glyph class names  \    backslash       Identifies CIDs. Distinguishes glyph names from an identical keyword  -    hyphen          Denotes glyph ranges in a glyph class  =    equalsign       Glyph class assignment operator  '    single quote    Marks a glyph or glyph class for contextual substitution or positioning  " "  double quotes   Enclose a name table string  { }  braces          Enclose a feature, lookup, table, or anonymous block  [ ]  square brackets Enclose components of a glyph class  < >  angle brackets  Enclose a device, value record, contour point, anchor, or caret  ( )  parentheses     Enclose the file name to be included2.c.
