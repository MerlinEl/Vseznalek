When InDesign is next launched it will create new preference files and the program will be restored to its defaults.same issue here, postscript file generated from print was 33.9 MB before deleting the preference file, after 1.47 GB.Deleted the Version 13.0 folder and restarted indesign./Users/user/Library/Preferences/Adobe InDesign/Version 13.0
